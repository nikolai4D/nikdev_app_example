This is a demo project to learn the way Nikdev build its apps.

The stack is:
- node.js
- npm
- node >= v14
- express.js
- vite (might change)
- ContechOS as an external database.

- nd_frontend_framework -> nikdev frontend framework
- more nikdev packages to come.
